설치

사용하는 브라우저에 맞게 설치하기

데스크톱에서 쓰는 브라우저를 선택해 DictionEvery를 설치하세요. 설치 후 웹페이지에서 단어를 선택하면 뜻 확인, 저장, 복습까지 한 흐름으로 이어집니다.

Chrome

Chrome에서 가장 먼저 체험하기 좋은 버전

Chrome에서 단어 선택 버튼, 사전 패널, 단어장, 통합 테스트를 모두 사용할 수 있습니다. 데스크톱 웹에서 DictionEvery를 시작하기 좋은 선택입니다.

Chrome용 다운로드

Microsoft Edge

Microsoft Edge에 맞춘 버전

Edge에서도 웹페이지의 단어를 바로 찾고 저장할 수 있습니다. Windows 데스크톱 환경에서 익숙한 브라우저 흐름 그대로 사용할 수 있습니다.

Edge용 다운로드

Samsung Browser for Windows

Samsung Browser for Windows용 버전

Windows용 Samsung Browser에서 웹문서와 학습 자료를 읽을 때 단어를 바로 찾고 저장할 수 있도록 준비한 버전입니다.

Samsung Browser용 다운로드

Safari on macOS

macOS Safari용 버전

Mac에서 Safari를 주 브라우저로 쓰는 사용자를 위한 버전입니다. 웹페이지를 읽으며 모르는 단어를 DictionEvery 패널로 바로 정리할 수 있습니다.

Safari용 다운로드

Firefox

Firefox 데스크톱용 버전

Firefox 사용자를 위해 따로 준비한 버전입니다. 브라우저를 바꿔도 단어 저장, 뜻 확인, 복습 흐름은 같은 방식으로 이어집니다.

Firefox용 다운로드

Yandex Browser

Yandex Browser 데스크톱용 버전

Yandex Browser에서 읽는 중 선택한 단어를 빠르게 사전과 단어장으로 연결합니다. 데스크톱 웹 독해 흐름에 맞춰 사용할 수 있습니다.

Yandex Browser용 다운로드

Naver Whale

Naver Whale용 버전

Whale에서 웹문서와 학습 자료를 읽을 때 DictionEvery를 함께 사용할 수 있도록 준비한 버전입니다. 뜻 확인과 복습 기능을 같은 흐름으로 제공합니다.

Whale용 다운로드

작동 방식

읽으면서 바로 저장하고 복습합니다

1

단어 선택

데스크톱 웹페이지에서 단어를 선택하면 작은 검색 버튼이 뜹니다.

2

도착언어별 뜻 확인

한국어, 영어, 러시아어, 페르시아어 등 선택한 언어별로 뜻과 예문을 확인합니다.

3

단어장과 테스트

숙련도 1-5를 관리하고 카드, 뜻 고르기, 듣고 입력하기를 섞어 복습합니다.
